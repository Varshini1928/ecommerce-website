E-Commerce Website

This project is a simple and responsive E-commerce Website built using HTML, CSS, and JavaScript.

The website allows users to browse products, view product details, add items to cart, and manage their shopping cart.

Features

Responsive homepage

Product listing section

Product details page

Add to cart functionality

Remove from cart

Cart total calculation

Simple and clean UI design

Technologies Used

HTML – Structure of the website

CSS – Styling and layout

JavaScript – Functionality and interactivity

Project Structure
ecommerce-website/
│
├── index.html        (Homepage)
├── style.css         (Stylesheet)
├── script.js         (JavaScript logic)
└── images/           (Product images)

How to Run

Download or clone the project

Open the project folder

Double click on index.html

The website will open in your browser

(No server required)

Functionalities

Users can browse products

Add products to cart

View cart items

Calculate total price

Remove items from cart

Future Improvements

Database support (MySQL / MongoDB)
